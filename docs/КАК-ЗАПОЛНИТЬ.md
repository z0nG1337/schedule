# Как заполнить таблицу (без опыта)

## Шаг 1. Скачайте шаблон

Файл: `docs/prostyj-shablon.csv` — откройте в блокноте или импортируйте в Google Таблицы.

## Шаг 2. Импорт в Google

1. https://sheets.google.com → **Создать** → **Пустая таблица**
2. **Файл** → **Импорт** → вкладка **Загрузка**
3. Выберите `prostyj-shablon.csv`
4. **Импортировать**

## Шаг 3. Меняйте только данные

Первая строка **не трогайте** (заголовки).

Дальше — как в примере:

| День недели | Время | Предмет | Кабинет | Учитель | Группа |
|-------------|-------|---------|---------|---------|--------|
| понедельник | 08:30 - 09:15 | Математика | 101 | Иванов | 10А |
| *(пусто)* | 09:30 - 10:15 | Русский | 102 | Петров | 10А |

- Если тот же день — **«День недели» можно оставить пустым**
- **Группа** в каждой строке (10А, 10Б…)
- Новый день — снова пишите `вторник`, `среда`…

## Шаг 4. Опубликовать

1. **Файл** → **Поделиться** → **Опубликовать в интернете**
2. **Опубликовать**
3. Скопировать ссылку (с `output=csv`)

## Шаг 5. Vercel

Переменная `GOOGLE_SHEETS_CSV_URL` = ваша ссылка → **Redeploy**.

Сайт: https://school-schedule-beige.vercel.app/student
